# alltoall_perf：out-of-place busbw 对比

- **指标**：**out-of-place** 的 **busbw（GB/s）**；每个 message size 为 **3 次 run 的算术平均**。
- **配置**：**AR OFF** | **AR ON DDP OFF** | **AR ON DDP ON**（对应日志 CASE1 / CASE2 / CASE3；“DDP” 与日志中 CASE3 的 IB 特性组合一致）。
- **百分比差**：Δ% = (该列 − AR OFF) / AR OFF × 100%。
- **环境**：`NCCL_P2P_DISABLE=1`，`alltoall_perf -b 1k -e 8G -f 2 -g 1`，16 ranks，Docker/工作区路径以原日志为准。
