# all_reduce_perf：out-of-place busbw 对比

- **数据来源**：`all_reduce_perf` 日志，环境含 `NCCL_P2P_DISABLE=1`（P2P 关闭），2×8 GPU（16 ranks）。
- **指标**：**out-of-place** 列的 **busbw（GB/s）**；每个 message size 对 3 次 run 取算术平均。
- **对比配置**  
  - **AR OFF**（CASE1）  
  - **AR ON DDP OFF**（CASE2，`NCCL_IB_ADAPTIVE_ROUTING=1`）  
  - **AR ON DDP ON**（CASE3，自适应路由 + OOO/匹配/预投递等 DDP 相关开关为 ON）
- **百分比差**：\(\Delta\% = (\text{该列} - \text{AR OFF}) / \text{AR OFF} \times 100\%\)。
